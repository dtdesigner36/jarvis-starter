# Tier 2: blog

**Stack:** Astro (content-focused), Next.js + MDX, or Gatsby
**Key files:** content/, posts/, rss.xml
**Skills:** `/frontend-design`, `/theme-token`, `/responsive-check`
**Wiki folders:** Posts/, Templates/, SEO/
**Triggers:**
- New post → check frontmatter (slug, date, tags)
- Image added → check optimization (webp, sizes)
**Pitfalls:**
- No RSS feed
- No sitemap.xml
- Images not optimized (size, format)
- No OG image generation
- Slow Time-to-First-Byte on static
