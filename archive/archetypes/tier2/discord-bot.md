# Tier 2: discord-bot

**Stack:** Node + discord.js, or Python + discord.py
**Key files:** commands/, events/, main entry
**Skills:** `/new-system`, `/devlog`
**Wiki folders:** Commands/, Events/, Integrations/
**Triggers:**
- New command/event → wiki/Commands|Events/
- Bot token in code → warn
**Pitfalls:**
- Intents not set correctly
- No error handling in handlers
- Discord API rate limits
